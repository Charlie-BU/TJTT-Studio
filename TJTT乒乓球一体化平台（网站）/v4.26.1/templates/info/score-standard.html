{{ _("TJTT团体积分说明")}}

{{_("TJTT首创积分赛制")}}

序号 说明
1 代表队暂以学院、学堂为单位。考虑到不同赛事面对对象有所差异，每个代表队与相应团体赛事绑定，不同赛事中代表队不同。但每位运动员可同时从属多个代表队。
2 每个代表队有队员人数上限。且队内积分排名前三的队员为主力队员，在代表队积分计算中具有较大权重。
3 代表队（队员数=n）积分计算公式：
n = 0：代表队积分 = 0
0 < n ≤ 3：代表队积分 = 各队员个人积分和 / n
n ≥ 4：代表队积分 = 3名主力个人积分和 / 3 × 0.6 + 其余队员个人积分和 / (n-3) × 0.4
4 选手报名团体赛时，需如实选择参赛代表队。若虚假报名其他代表队，将视情况给予相应处罚。报名取消后选手重新报名，上一次报名信息会被覆盖，比赛进行以最后一次报名信息为准。
5 团体赛事中，单打比赛积分正常计算；双打比赛以队员积分均值作为双方积分进行计算，计算规则与单打相同。结算时胜方两位选手均加相应积分，负方两位选手均减相应积分。
6 比赛结束后各选手个人积分与各代表队积分将自动同步至“积分排名”中。
